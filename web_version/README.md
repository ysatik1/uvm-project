# Веб-версия Учебной Виртуальной Машины

## Использование

Просто откройте файл `index.html` в браузере.

## Возможности

- Полный интерпретатор УВМ на JavaScript
- Редактор кода с подсветкой синтаксиса
- Вывод выполнения в реальном времени
- Таблица памяти с просмотром значений
- Визуализация стека
- Примеры программ

## Поддерживаемые команды

1. `LOAD_CONST <value>` - загрузка константы
2. `READ_MEM <address>` - чтение из памяти
3. `WRITE_MEM <address>` - запись в память
4. `SGN <address>` - вычисление знака

## Запуск

1. Откройте `index.html` в любом современном браузере
2. Напишите или выберите пример программы
3. Нажмите "Выполнить"
4. Смотрите результат во вкладках вывода

Включены примеры программ для тестирования всех возможностей УВМ.
